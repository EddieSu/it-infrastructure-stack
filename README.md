# Self-Hosted Enterprise IT Infrastructure — Tech Stack Showcase

A bilingual (EN / 繁中) technology-selection showcase of a one-person-operated
enterprise IT infrastructure: virtualization, container hosting, custom-built
applications, self-hosted business services, backup, observability, mail relay,
and an on-prem AI/RAG stack.

本站是一份**中英雙語**的技術選型展示，呈現一人維運的自架企業 IT 基礎架構：虛擬化、
容器主機、自寫應用、自架商業服務、備份、可觀測性、郵件中繼，以及地端 AI/RAG 棧。

> **Showcase only.** No internal IP addresses, hostnames, internal DNS,
> credentials, or product branding appear anywhere in this site — only
> *what kind of system* and *what technology combination* powers it.
>
> **純展示。** 全站不出現任何內部 IP、主機名、內部 DNS、帳密或產品品牌，
> 只講「這是什麼類型的系統」與「用了什麼技術組合」。

## Built with
